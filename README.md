# 📊 Sales Forecasting Dashboard

## 📌 Project Overview
This project focuses on analyzing historical sales data and predicting future sales trends using data visualization and forecasting techniques. The dashboard provides clear insights into sales performance and future growth patterns.

---

## 🎯 Objective
To build a sales forecasting model that helps in understanding past trends and predicting future sales for better business decision-making.

---

## 🛠️ Tools Used
- Microsoft Excel
- Microsoft Power BI

---

## 📊 Features
- Data cleaning and preparation using Excel  
- Interactive dashboard using Power BI  
- Sales trend visualization  
- Forecasting future sales using analytics tools  
- KPI indicators:
  - Total Sales
  - Average Sales  

---

## 📈 Dashboard Insights
- Sales show a steady increasing trend over time  
- Forecast indicates continued growth in future sales  
- Helps businesses plan inventory and strategy  

---

## 📁 Files Included
- Dataset (Excel file)  
- Power BI Dashboard (PDF/Image)  
- Project Description  

---

## 🚀 Conclusion
This project demonstrates how data visualization and forecasting can be used to extract meaningful insights from business data. It helps in predicting future sales and supports effective decision-making.
